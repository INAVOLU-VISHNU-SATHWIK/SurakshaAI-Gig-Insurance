# SurakshaAI - AI-Powered Insurance for Gig Workers 🚀

## 📌 Problem Statement

Gig workers such as delivery partners (Swiggy, Zomato, Amazon, etc.) often face income loss due to external disruptions like heavy rain, extreme heat, pollution, or sudden curfews. Currently, there is no structured system to compensate for these uncontrollable events.

## 💡 Our Solution

**SurakshaAI** is an AI-powered parametric insurance platform designed to protect gig workers from income loss caused by external disruptions.

Instead of traditional insurance, our system automatically detects real-world events (like heavy rain) and triggers instant payouts without requiring manual claims.

---

## 🎯 Key Features

### 🔹 AI-Based Risk Assessment

* Dynamic premium calculation based on location and risk factors
* Predictive analysis using environmental data

### 🔹 Weekly Pricing Model

* Affordable weekly subscription model
* Designed specifically for gig workers' income cycle

### 🔹 Parametric Insurance Triggers

* Automatic payout when predefined conditions are met:

  * Heavy Rain
  * Extreme Heat
  * High Pollution Levels
  * Curfews / Area Restrictions

### 🔹 Intelligent Fraud Detection

* Detects abnormal claim patterns
* Location validation
* Prevents duplicate claims

### 🔹 Automated Claim & Payout

* Zero manual claim process
* Instant payout simulation

---

## 👤 Target Persona

We focus on **Food Delivery Partners (Swiggy/Zomato)** who depend on daily earnings and are highly affected by weather disruptions.

---

## ⚙️ How It Works (Workflow)

1. User registers on the platform
2. Selects a weekly insurance plan
3. System calculates premium using AI-based risk analysis
4. Real-time data (weather/conditions) is monitored
5. If a disruption occurs:

   * System automatically triggers a claim
   * Payout is processed instantly

---
